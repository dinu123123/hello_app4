Welcome to FinAdmin

The Menu of FinAdmin consists of the following items needed to be introduced:

  • Trucks
    • New Truck - Create a new truck record in the DB
    • All Trucks - View all existent trucks present in the DB
  • Trucks Expenses
    • New Truck Expense - Create a new truck expense (expense related to the repair and mantenance of the vehicule) record in the DB
    • All Truck Expenses - View all existent truck expenses (with their specific types) present in the DB
  • Driver Expenses
    • New Driver Expense - Create a new driver expense (expense related payments made by the driver) record in the DB
    • All Trucks - View all existent driver expenses (with their specific types) present in the DB
  • Toll Generic
    • New Toll Generic - Create a new generic toll expense (Number plate of the truck it applies, period of concern (Start/End date),
      covered distance and cost in Euro currency) and record it in the DB
    • All Trucks - View all existent generic toll expenses present in the DB
  • Driver Events
    • New Driver Events - Create a new driver event (to specify the date a which a driver starts/stops working on/for a truck/client)
      and record it in the DB
    • All Events - View all existent driver events
  • Fuel Expense
    • New Fuel Expense - Create a new fuel expense specification
    • File Import - Import fuel expense specification as provided by IDS csv file
    • All Fuel Expenses - View all existent fuel expenses made within a time interval
  • Toll Germany
    • New Toll Germany - Create a new toll Germany specification
    • File Import - Import Germany toll specification as provided by TollCollect csv file
    • All Fuel Expenses - View all existent Germany toll costs made within a time interval
  • Toll Belgium
    • New Toll Belgium - Create a new toll Belgium specification
    • File Import - Import Belgium toll specification as provided by Satellic csv file
    • All Fuel Expenses - View all existent Belgium toll costs made within a time interval
